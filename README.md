# 📊 E-Commerce Sales Dashboard (Power BI & Tableau)

---

### 🔍 Project Overview

This project analyzes e-commerce sales data to uncover actionable business insights using Power BI and Tableau. The dashboards help visualize sales performance across regions, customer segments, and time, enabling better decision-making.

---

## 🎯 Problem Statement

E-commerce businesses often face challenges in:

- Identifying high-performing regions and customer segments.
- Understanding sales trends and seasonality.
- Making data-driven decisions to improve revenue.

This project addresses these challenges by building interactive dashboards that transform raw data into meaningful insights.

---

## 📊 Objectives

- Analyze sales distribution across regions and segments.
- Identify trends and patterns over time.
- Highlight top-performing categories/products.
- Enable quick business decision-making through visualization.

---

## 🛠️ Tools & Technologies

- Power BI
- Tableau
- MySQL
- Python (Jupyter Notebook)
- Microsoft Excel
- Data Visualization & Analysis

---

## 📁 Dataset Description

The dataset contains:

- Order Date
- Sales
- Region
- Segment
- Product Name
- Category / Sub-category

- Sales by Category/Product

---

## 💡 Business Insights

- 📌 The West region generates the highest sales.
- 📌 The Consumer segment contributes the majority (~49%) of total revenue.
- 📌 Sales show noticeable fluctuations with peaks in certain periods.
- 📌 Some regions and segments underperform, indicating opportunities for improvement.

---

## 🚀 Features

- Interactive dashboards.
- Cross-tool implementation (Power BI + Tableau).
- Clear visualization of trends and patterns.
- Comparative analysis across regions and segments.
- End-to-end analysis (Excel → SQL → Python → BI tools).
  
---

## 📌 How to Use

- Open the Power BI file (.pbix) in Power BI Desktop.
- Open the Tableau file (.twbx) in Tableau.
- Explore the dashboards and interact with filters.
