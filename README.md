# 📊 E-Commerce Sales Dashboard (Power BI & Tableau)
---
## 🔍 Project Overview
This project focuses on analyzing e-commerce sales data to uncover key business insights using Power BI and Tableau. The dashboards provide a clear understanding of sales performance across regions, customer segments, and time.
---
## 🎯 Problem Statement
Businesses often struggle to:

- Identify top-performing regions and customer segments
- Understand sales trends over time
- Make data-driven decisions for growth

This project aims to solve these challenges by creating interactive dashboards for better decision-making.
---
## 🛠️ Tools & Technologies
- Power BI
- Tableau

- Data Analysis
- MySQL
- Jupyter Notebook
- Microsoft Excel
---
## 📁 Dataset Description
The dataset contains:

- Order Date

- Product Details

---
## 💡 Business Insights
-📌 The West region generates the highest sales.
-📌 The Consumer segment contributes the majority (~49%) of total revenue.
-📌 Sales show noticeable fluctuations with peaks in certain periods.
-📌 Some regions and segments underperform, indicating opportunities for improvement.
---
## 🚀 Features
- Interactive dashboards.
- Clear visualization of trends and patterns.
- Comparative analysis across regions and segments.
---
## 📌 How to Use
- Open the Power BI file (.pbix) in Power BI Desktop.
- Open the Tableau file (.twbx) in Tableau.
- Explore the dashboards and interact with filters.
